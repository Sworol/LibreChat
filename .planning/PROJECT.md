# LibreChat

## What This Is

LibreChat fork with monetization layer for Chinese users.

## Core Value

API resale platform targeting Chinese users who can't use foreign AI services directly.

## Current Milestone: v1.1 变现版本

**Goal:** Add complete monetization system to LibreChat

**Target features:**
- 积分/余额系统
- 微信/支付宝充值
- 看广告送积分
- 会员等级体系
- API 调用计费
